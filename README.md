### Current Project: Gaffer's Clipboard
My primary development focus is **Gaffer's Clipboard**, a high-fidelity desktop companion application for EA FC / FIFA Career Mode. This project acts as a practical bridge between my data science studies and my interest in sports analytics, utilizing a strict, interface-driven MVC architecture to manage deep simulations across the Premier League, Championship, La Liga, Eredivisie, and Bundesliga.

* **Custom OCR Engine:** Developed a lightweight, resolution-independent OpenCV (KNN) computer vision pipeline that extracts team and player stats from screenshots in seconds, bypassing the need for heavy ML frameworks.
* **Advanced Analytics Engine:** Currently implementing an analytics suite that uses Principal Component Analysis (PCA) for custom match ratings, Exponential Moving Averages for form scores, Random Forest for win-condition extraction, and K-means clustering for match grouping.
* **Strict Data Validation:** Utilizes strict Pydantic V2 models to ensure all extracted data is mathematically and logically sound before committing it to a JSON-backed local database.
* **Rigorous Engineering Standards:** The application adheres to strict MVC practices and utilizes industry-standard tools like Astral Software's `ruff`, `ty`, and `uv`. I am also actively building out comprehensive test coverage through a suite of `pytest` tests.
